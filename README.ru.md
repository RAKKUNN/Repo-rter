Красивый инструмент аналитики трафика GitHub в нео-брутальном/пиксельном стиле, созданный с помощью Tauri и Next.js.

<img src="./logo.png" width="100" />

## Возможности
- 📈 Бесконечное отслеживание трафика GitHub (обход 14-дневного ограничения)
- 🎨 Нео-брутальный / Пиксельный UI
- 🔄 Десктопное приложение с треем и фоновой синхронизацией
- 💾 Экспорт данных в CSV и JSON
- 🌐 Поддержка 12 языков

## Начало работы

Для запуска приложения локально:

```bash
npm install
npm run tauri dev
```

## 📖 О Repo-rter
GitHub предоставляет данные только за 14 дней. Repo-rter сохраняет их бесконечно.

## 🚀 Возможности
- 📈 Бесконечное отслеживание
- 🎨 Пиксельный UI
- 🔄 Фоновая синхронизация
- 💾 Экспорт в CSV и JSON
- 🌐 Поддержка 12 языков

## 🛠️ Стек
- **Frontend**: Next.js
- **Backend**: Tauri

## 💻 Локальная разработка

Для запуска:

```bash
npm install
npm run tauri dev
```

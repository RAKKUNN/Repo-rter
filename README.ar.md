أداة جميلة لتحليل زيارات GitHub بواجهة بكسل / نيو بروتاليزم، مبنية باستخدام Tauri و Next.js.

<img src="./logo.png" width="100" />

## الميزات
- 📈 تتبع زيارات GitHub بلا حدود (تجاوز حد 14 يومًا)
- 🎨 واجهة مستخدم بكسل / نيو بروتاليزم
- 🔄 تطبيق سطح مكتب مع علبة نظام ومزامنة في الخلفية
- 💾 تصدير البيانات إلى CSV و JSON
- 🌐 يدعم 12 لغة

## البدء

لتشغيل التطبيق محليًا:

```bash
npm install
npm run tauri dev
```

## 📖 حول Repo-rter
يوفر GitHub بيانات لمدة 14 يومًا فقط. تعمل هذه الأداة في الخلفية لحفظ بياناتك بلا حدود.

## 🚀 الميزات
- 📈 تتبع زيارات بلا حدود
- 🎨 واجهة مستخدم بكسل
- 🔄 مزامنة في الخلفية
- 💾 تصدير إلى CSV و JSON
- 🌐 يدعم 12 لغة

## 🛠️ التكنولوجيا
- **Frontend**: Next.js
- **Backend**: Tauri

## 💻 التطوير المحلي

لتشغيل التطبيق:

```bash
npm install
npm run tauri dev
```

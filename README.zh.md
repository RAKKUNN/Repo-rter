一个美丽、新粗野主义/像素艺术风格的 GitHub 流量分析工具，使用 Tauri 和 Next.js 构建。

<img src="./public/logo.png" width="100" />

## 功能
- 📈 无限跟踪 GitHub 流量（绕过 14 天限制）
- 🎨 新粗野主义 / 像素艺术 UI
- 🔄 带有系统托盘和后台同步的桌面应用
- 💾 将数据导出为 CSV 和 JSON
- 🌐 支持 12 种语言

## 入门

在本地运行应用程序：

```bash
npm install
npm run tauri dev
```

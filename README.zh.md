## 📖 关于 Repo-rter
GitHub 仅提供 14 天的流量数据。Repo-rter 在后台持续运行，无限期地跟踪并将数据保存在本地。

## 🚀 功能
- 📈 无限跟踪 GitHub 流量（绕过 14 天限制）
- 🎨 新粗野主义 / 像素艺术 UI
- 🔄 带有系统托盘和后台同步的桌面应用
- 💾 将数据导出为 CSV 和 JSON
- 🌐 支持 12 种语言

## 🛠️ 技术栈
- **前端**: Next.js, React, TailwindCSS
- **后端**: Tauri (Rust)

## 💻 本地开发

在本地运行应用程序：

```bash
npm install
npm run tauri dev
```

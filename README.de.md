Ein wunderschönes GitHub-Traffic-Analyse-Tool.

<img src="./public/logo.png" width="100" />

## 📖 Über Repo-rter
GitHub bietet nur 14 Tage Daten. Repo-rter speichert Ihre Daten unbegrenzt.

## 🚀 Funktionen
- 📈 Unbegrenztes Tracking
- 🎨 Neo-Brutalismus UI
- 🔄 Hintergrund-Synchronisation
- 💾 CSV Export
- 🌐 12 Sprachen

## 🛠️ Technologie
- **Frontend**: Next.js
- **Backend**: Tauri

## 💻 Entwicklung

So starten Sie:

```bash
npm install
npm run tauri dev
```

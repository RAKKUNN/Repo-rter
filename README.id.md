## 📖 Tentang Repo-rter
GitHub hanya menyediakan 14 hari data. Repo-rter menyimpannya tanpa batas.

## 🚀 Fitur
- 📈 Pelacakan tanpa batas
- 🎨 UI Pixel-art
- 🔄 Sinkronisasi latar belakang
- 💾 Ekspor CSV
- 🌐 12 bahasa

## 🛠️ Teknologi
- **Frontend**: Next.js
- **Backend**: Tauri

## 💻 Memulai

### Prasyarat
- Node.js (v18+)
- Rust

### Instalasi
```bash
git clone https://github.com/RAKKUNN/Repo-rter.git
cd Repo-rter
npm install
npm run tauri dev
```

## 📖 Tentang Repo-rter
GitHub hanya menyediakan 14 hari data. Repo-rter menyimpannya tanpa batas.

## 🚀 Fitur
- 📈 Pelacakan tanpa batas
- 🎨 UI Pixel-art
- 🔄 Sinkronisasi latar belakang
- 💾 Ekspor CSV
- 🌐 12 bahasa

## 🛠️ Teknologi
- **Frontend**: Next.js
- **Backend**: Tauri

## 💻 Memulai

Untuk menjalankan:

```bash
npm install
npm run tauri dev
```

## 📖 Sobre Repo-rter
O GitHub fornece apenas 14 dias de dados. O Repo-rter guarda os seus dados infinitamente.

## 🚀 Características
- 📈 Rastreamento infinito
- 🎨 UI Neo-brutalista
- 🔄 Sincronização em segundo plano
- 💾 Exportar para CSV
- 🌐 Suporta 12 idiomas

## 🛠️ Tecnologia
- **Frontend**: Next.js
- **Backend**: Tauri

## 💻 Desenvolvimento

Para rodar:

```bash
npm install
npm run tauri dev
```

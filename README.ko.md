Tauri와 Next.js로 구축된 아름답고 직관적인 네오 브루탈리즘 / 픽셀 아트 스타일의 깃허브 트래픽 분석 툴입니다.

<img src="./logo.png" width="100" />

## 주요 기능
- 📈 깃허브 트래픽 무제한 기록 (14일 제한 우회)
- 🎨 네오 브루탈리즘 / 픽셀 아트 UI
- 🔄 시스템 트레이 및 크론(Cron)을 통한 백그라운드 데이터 수집
- 💾 CSV 및 JSON 데이터 내보내기
- 🌐 12개 언어 지원

## 시작하기

로컬에서 앱 실행하기:

```bash
npm install
npm run tauri dev
```

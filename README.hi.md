## 📖 Repo-rter के बारे में
गिटहब केवल 14 दिनों का डेटा देता है। Repo-rter बैकग्राउंड में चलता है और आपके डेटा को असीमित रूप से सहेजता है।

## 🚀 विशेषताएं
- 📈 ट्रैफ़िक को असीम रूप से ट्रैक करें
- 🎨 पिक्सेल-आर्ट यूआई
- 🔄 बैकग्राउंड सिंकिंग
- 💾 CSV और JSON में निर्यात करें
- 🌐 12 भाषाओं का समर्थन

## 🛠️ तकनीक
- **Frontend**: Next.js
- **Backend**: Tauri

## 💻 शुरू करना

ऐप चलाने के लिए:

```bash
npm install
npm run tauri dev
```

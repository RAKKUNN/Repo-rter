## 📖 Repo-rter について
GitHubは14日間のデータしか提供しません。Repo-rterはデータをローカルに無期限で保存します。

## 🚀 機能
- 📈 無制限のトラフィック追跡
- 🎨 ピクセルアートUI
- 🔄 バックグラウンド同期
- 💾 CSV / JSON エクスポート
- 🌐 12言語対応

## 🛠️ 技術スタック
- **Frontend**: Next.js
- **Backend**: Tauri

## 💻 開発

ローカルで実行するには：

```bash
npm install
npm run tauri dev
```

TauriとNext.jsで構築された、美しくネオブルータリズム/ピクセルアートのGitHubトラフィック分析ツール。

<img src="./logo.png" width="100" />

## 機能
- 📈 GitHubトラフィックを無制限に追跡（14日間の制限を回避）
- 🎨 ネオブルータリズム / ピクセルアート UI
- 🔄 システムトレイとバックグラウンド同期を備えたデスクトップアプリ
- 💾 データをCSVおよびJSONにエクスポート
- 🌐 12言語をサポート

## はじめに

アプリをローカルで実行するには：

```bash
npm install
npm run tauri dev
```
